# Simple CLI Calculator
A simple command-line calculator in Python supporting basic arithmetic operations (+, -, *, /, %).

A beginner-friendly command-line calculator built using Python.
This project supports basic arithmetic operations and helps reinforce Python fundamentals such as input handling, conditionals, and operators.

Features

Addition (+)
Subtraction (-)
Multiplication (*)
Division (/)
Modulus (%)
Command-line interface
